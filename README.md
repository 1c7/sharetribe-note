## Tech Stack 技术栈
官方的 readme 里面只是简单说了下安装和配置，和一些基本的依赖。  
没有列清楚技术栈。所以我这里一边看代码学习一边列出来技术栈  

Ruby 2.3.1  
Ruby on Rails 4.2.7  
Gem: Devise(做邮箱登录), Omniauth-facebook(做 Facebook 登录)  
MySQL  

服务器：不清楚，可能是 AWS  
图片托管：不清楚，可能是 CloudFront
