## Tech Stack 技术栈
官方的 readme 里面只是简单说了下安装和配置。  没有列清楚技术栈。
所以我这里看代码列出来技术栈

Ruby 2.3.1  
Ruby on Rails 4.2.7  
Gem: Devise(做邮箱登录), Omniauth-facebook(做 Facebook 登录)
